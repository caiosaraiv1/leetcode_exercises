# 🚀 LeetCode Challenges (C++)

Este repositório foi criado para documentar e organizar a minha jornada de resolução de problemas no **LeetCode**, utilizando **C++**. O objetivo principal é consolidar o domínio de estruturas de dados, otimização de código e padrões clássicos de algoritmos (DSA).

A estrutura abaixo foi desenhada de forma modular e categorizada por **padrões de design de algoritmos**, facilitando a expansão à medida que novos desafios forem resolvidos.

---
